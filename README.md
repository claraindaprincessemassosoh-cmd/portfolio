# E-Portfolio — Clarainda Massosoh

E-Portfolio développé avec **Symfony 7** dans le cadre du BUT Réseaux & Télécommunications à l'IUT de Roanne.

## Pages

- **Accueil** — Présentation, compétences RT1/RT2/RT3, technologies
- **Compétences** — Apprentissages critiques du référentiel BUT R&T
- **Hobbies** — Centres d'intérêt personnels
- **Contact** — Coordonnées et formulaire de contact

## Installation

```bash
# 1. Cloner le dépôt
git clone https://github.com/claraindaprincessemassosoh-cmd/portfolio.git
cd portfolio

# 2. Installer les dépendances
composer install

# 3. Lancer le serveur
symfony server:start
```

Ouvrir ensuite [http://localhost:8000](http://localhost:8000)

## Stack technique

- PHP 8.1+
- Symfony 7
- Twig (templates)
- CSS vanilla (dark theme)
- Tabler Icons
